# S.O.L.I.D

![PHP](https://img.shields.io/badge/php-%23777BB4.svg?style=for-the-badge&logo=php&logoColor=white) ![Composer](https://img.shields.io/badge/Composer-885630?style=for-the-badge&logo=composer&logoColor=white) ![SOLID](https://img.shields.io/badge/solid-2C4F7C?style=for-the-badge&logo=solid&logoColor=white)

SOLID é um conjunto de 5 princípios de design que tornam o código mais organizado, flexível e fácil de manter. Neste estudo, estou aplicando SOLID no desenvolvimento com PHP.



### Ferramentas

Composer é o gerenciador de dependências padrão do PHP. Ele facilita a instalação, atualização e autoload de bibliotecas, promovendo organização, reaproveitamento de código e padronização do projeto. Estou utilizando o Composer para estruturar melhor meus estudos em PHP e aplicar boas práticas no gerenciamento de pacotes.
